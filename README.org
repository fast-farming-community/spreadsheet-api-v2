* FastApi

** Local Development

- Create a ~.env~ file containing:

  - ~GOOGLE_APPLICATION_CREDENTIALS~
  - ~DATABASE_URL~
  - ~SECRET_KEY_BASE~
  - ~PATREON_API_KEY~
  - ~PATREON_CAMPAIGN~
  - `GOOGLE_APPLICATION_CREDENTIALS`: `$HOME/.config/gcloud/configurations/config_default.json`

- Source the ~.env~ file
- Start the local environment ~task local-env~
- Create the ~fast_api_dev~ or ~fast_api_test~ database using the PgAdmin UI
- Create the tables ~task migrate~
- Seed the database ~task seed~
- Start the application ~task run-dev~
~ Run ~FastApi.Sync.GoogleSheetsUpdater.execute(FastApi.Schemas.Fast.Table)~ to fetch the rows
